# Learning & Robotics Notes

This is the central repository for technical documentation, devlogs, and research summaries. Storing these in the portfolio repo allows for easy publishing and "Learning in Public."

## Structure

- **/systems**: Modern C++, Firmware optimization (STM32/ESP32), RTOS, and performance engineering.
- **/control**: The math of robotics. Kinematics, Dynamics, PID, and State Estimation.
- **/ai**: Robot learning, Imitation Learning (LeRobot), and Computer Vision.
- **/devlog**: Chronological logs of project progress, bugs fixed, and daily wins.

## Habit: The 5-Minute Log
At the end of every coding session, create a new entry in `/devlog` or update the current project log.
1. **Context**: What was I working on?
2. **Breakthrough/Fix**: What did I learn or solve?
3. **Next Steps**: Where do I start tomorrow?
